### 七、热重载机制

```java
public class ConfigManager {
    // 文件监视器
    private WatchService watchService;
    
    public void watchFile(String path) {
        Path configPath = Paths.get(path);
        watchService = configPath.getFileSystem().newWatchService();
        configPath.getParent().register(watchService, 
            StandardWatchEventKinds.ENTRY_MODIFY);
        
        // 后台线程监视
        executor.submit(() -> {
            while (running) {
                WatchKey key = watchService.take();
                for (WatchEvent<?> event : key.pollEvents()) {
                    if (event.kind() == ENTRY_MODIFY) {
                        // 重新加载配置
                        reload();
                    }
                }
                key.reset();
            }
        });
    }
    
    public void reload() {
        // 清除缓存
        cache.clear();
        
        // 重新加载所有源
        for (ConfigSource source : sources.values()) {
            if (source instanceof Reloadable) {
                ((Reloadable) source).reload();
            }
        }
        
        // 通知监听器
        listeners.forEach(l -> l.onReload());
    }
}
```

```java
package shenyf.p5engine.config;

public class IniParser {
    
    public static Map<String, String> parse(Path filePath) {
        Map<String, String> result = new LinkedHashMap<>();
        
        try (BufferedReader reader = Files.newBufferedReader(filePath)) {
            String currentSection = "";
            String line;
            
            while ((line = reader.readLine()) != null) {
                line = line.trim();
                
                // 跳过空行和注释
                if (line.isEmpty() || line.startsWith(";") || line.startsWith("#")) {
                    continue;
                }
                
                // 节处理
                if (line.startsWith("[") && line.endsWith("]")) {
                    currentSection = line.substring(1, line.length() - 1);
                    continue;
                }
                
                // 键值对处理
                int eqIndex = line.indexOf('=');
                if (eqIndex > 0) {
                    String key = line.substring(0, eqIndex).trim();
                    String value = line.substring(eqIndex + 1).trim();
                    
                    // 移除引号
                    value = trimQuotes(value);
                    
                    // 构建带节的前缀key
                    String fullKey = currentSection.isEmpty() ? key : currentSection + "." + key;
                    result.put(fullKey, value);
                }
            }
        } catch (IOException e) {
            Logger.error("Failed to parse INI file: " + filePath, e);
        }
        
        return result;
    }
    
    private static String trimQuotes(String value) {
        if ((value.startsWith("\"") && value.endsWith("\"")) ||
            (value.startsWith("'") && value.endsWith("'"))) {
            return value.substring(1, value.length() - 1);
        }
        return value;
    }
}
```

```java
public class ConfigManager {
    
    /** 手动重载所有文件源 */
    public void reload() {
        Logger.info("Reloading configuration...");
        cache.clear();
        for (ConfigSource source : sources) {
            if (source instanceof Reloadable) {
                ((Reloadable) source).reload();
            }
        }
        Logger.info("Configuration reloaded");
    }
    
    /** 启用定时自动重载 */
    public void enableAutoReload(String filePath, long intervalMs) {
        disableAutoReload();
        watchService = Executors.newSingleThreadScheduledExecutor();
        watchService.scheduleAtFixedRate(() -> {
            ConfigSource source = getSource("file:" + filePath);
            if (source instanceof FileConfigSource) {
                if (((FileConfigSource) source).hasFileChanged()) {
                    reload();
                }
            }
        }, intervalMs, intervalMs, TimeUnit.MILLISECONDS);
        Logger.info("Auto-reload enabled for: " + filePath);
    }
    
    /** 禁用自动重载 */
    public void disableAutoReload() {
        if (watchService != null) {
            watchService.shutdown();
            watchService = null;
        }
    }
}
```

## Discoveries

1. **Processing 4 窗口管理变化**: Processing 4不再使用`PApplet.frame`字段，改用`PSurface`接口
2. **获取窗口的正确方式**: 
   - `PSurface surface = applet.getSurface()`
   - `surface.getNative()` 返回 `PSurfaceAWT$SmoothCanvas`
   - `SmoothCanvas.getFrame()` 返回 `JFrame`
   - 通过反射调用 `getFrame()` 方法获取窗口
3. **Processing CLI有真实窗口**: 不是无头模式，可以保存/恢复窗口位置
4. **组件需使用static内部类**: 因为反射创建组件时无法处理非静态内部类
5. **keyPressed每帧重置**: Processing的`keyPressed`在每帧`draw()`开始时重置，需用`registerMethod("keyEvent", this)`监听事件
