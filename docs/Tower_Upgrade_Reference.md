| 等级 | 攻击力加成 | 其他效果 | 升级花费 |
|------|-----------|---------|---------|
| 0 级 | — | — | — |

> 毒素效果为持续伤害（DoT），每 0.5 秒结算一次，最多叠加 **2 层**。满 2 层后新毒素会替换剩余时间最短的那一层。

## 附录：配置字段速查

| 配置字段 | 作用 | 适用塔 |
|---------|------|--------|
| `damageMult` | 伤害倍率 | MG、Missile |
| `bulletSizeMult` | 弹体/爆炸渲染尺寸倍率 | MG、Missile |
| `aoeMult` | 爆炸半径倍率 | Missile |
| `speedMult` | 冷却时间倍率（越小攻速越快） | Laser |
| `rangeMult` | 射程倍率 | Slow |
| `slowMult` | 减速系数倍率（越小减速越强） | Slow |
| `poisonMult` | 毒素伤害倍率 | Poison |
| `commandMult` | 攻击力加成倍率 | Command |
| `commandBonus` | 0 级攻击力加成倍率 | Command |
| `killBonusMin/Max` | 击杀奖金随机范围（0/1/2 级分别配置） | Command |
| `burnDamage` | 燃烧每秒伤害 | Missile（2 级） |
| `burnDuration` | 燃烧持续时间（秒） | Missile（2 级） |
| `poisonDamage` | 毒素每秒伤害 | Poison |
| `poisonDuration` | 毒素持续时间（秒） | Poison |
| `poisonFanAngle` | 扇形喷射角度（度） | Poison |
| `laserBonus` | 激光额外伤害 | Laser |
| `laserDelay` | 激光延迟命中时间（秒） | Laser |
| `slowFactor` | 减速系数 | Slow |
| `slowDuration` | 减速持续时间（秒） | Slow |
